2. Cara Menggunakan

Tahap 1: Login & Koneksi

1. Buka Dashboard & Pilih Nama Anda.

2. Lihat tampilan layar:

  • Jika Kotak Input HP Muncul: Artinya belum connect. Masukkan nomor WA > Klik "Dapatkan Kode" > Masukkan kode di HP.
  • Jika Kotak Input HP Hilang: Artinya WA sudah connect & siap pakai.

ILUSTRASI DI HP:

Titik 3 > Perangkat Tertaut > tautkan perangkat > tautkan dengan nomor telepon saja

Tahap 2: Eksekusi Blast

1. Klik tombol hijau START BLAST.

2. Jangan tutup tab browser (Boleh diminimize).

3. Pantau Log Terminal di bawah layar untuk memastikan proses berjalan.

FAQ / Bantuan

Web loading terus / Error 502?
Server mungkin sedang restart otomatis. Tunggu 1-2 menit, lalu Refresh halaman. Jika masih error lebih dari 10 menit, hubungi Admin IT.
Kode Pairing tidak muncul di layar?
Biasanya karena koneksi internet di HP tidak stabil saat meminta request ke server WhatsApp. Refresh halaman web, matikan Wifi HP (pakai data), lalu coba lagi.
Di Log "Sukses" tapi di HP gak ada chat?
Ini wajar. Sinkronisasi HP kadang delay (chat baru muncul 5 menit kemudian). Patokan utama adalah Google Sheet. Jika di Sheet statusnya sudah 'sent', berarti pesan sudah aman terkirim ke server WA.
